# Forum n' Chat

Web forum app with chat function between users run by Go server.

## Used Technologies

- html
- css
- js
- Go
  - standard packages
  - Gorilla websocket
  - sqlite3

## Installation & How to Use

This app is being hosted on Render. You can access https://forum-n-chat.onrender.com and start trying.
